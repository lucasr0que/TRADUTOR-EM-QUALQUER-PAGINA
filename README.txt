
TRADUÇÃO INSTANTÂNEA POR ATALHO

O TRADUTOR É UMA FERRAMENTA DE TRADUÇÃO RÁPIDA E EFICIENTE DESENVOLVIDA PARA SUA ÁREA DE TRABALHO. ELE TRANSFORMA QUALQUER TEXTO SELECIONADO EM UMA TRADUÇÃO INSTANTÂNEA, EXIBIDA DIRETAMENTE NA POSIÇÃO DO SEU MOUSE.

COMO UTILIZAR O PROGRAMA:

CONFIGURAÇÃO INICIAL: AO INICIAR O PROGRAMA, VOCÊ DEVE SELECIONAR A LINGUAGEM DE DESTINO DESEJADA (EXEMPLO: PORTUGUÊS). O PROGRAMA TRADUZIRÁ TUDO QUE IDENTIFICAR PARA ESSA LÍNGUA. HÁ 7 LINGUAGENS PRÉ-CONFIGURADAS DISPONÍVEIS.

SELECIONE O TEXTO: EM QUALQUER APLICATIVO OU ABA, SELECIONE O TEXTO QUE VOCÊ DESEJA TRADUZIR.

APERTE CTRL + Ç 

RESULTADO IMEDIATO: O PROGRAMA IDENTIFICA AUTOMATICAMENTE A LÍNGUA ORIGINAL E MOSTRA A TRADUÇÃO EM UM BALÃO FLUTUANTE NA POSIÇÃO EXATA DO SEU MOUSE.

PARA FECHAR: CLIQUE EM QUALQUER LUGAR DA TELA OU NA PRÓPRIA CAIXA DE TRADUÇÃO.

CASO QUEIRA MUDAR A LINGUAGEM ENQUANTO EXECUTA O PROGRAMA BASTA APERTAR : ALT + Ç 



PARA EXECUTAR ESSE PROGRAMA UTILIZE UMA VENV, NÃO INSTALE OS REQUERIMENTOS NO SEU PYTHON GLOBAL.(PARA QUEM VAI EXECUTAR NO TERMINAL)


COMO UTILIZAR O PROGRAMA:

1° OPÇÃO :

EXECUTAVEL: APENAS CLICAR 2 VEZES NO EXE E INICIAR

2° OPÇÃO :

NA PASTA DO PROGRAMA: ABRA O TERMINAL

NO CMD EXECUTE OS COMANDOS NA SEGUINTE ORDEM:

pip install -r requirements.txt
py CodigoFinal.py

NO POWERSHELL EXECUTE OS COMANDOS NA SEGUINTE ORDEM:

py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
py CodigoFinal.py

NO TERMINAL LINUX OU MACOS EXECUTE OS COMANDOS NA SEGUINTE ORDEM:

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 CodigoFinal.py

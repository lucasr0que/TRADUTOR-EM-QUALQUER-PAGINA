# TRADUTOR-BR
Esse tradutor serve para traduzir blocos de texto.

Inicialmente o programa vai te dar 7 linguagens ja pré configuradas, voce seleciona e o programa começa a rodar.

A interação com ele se da no momento que faz a seleçao de um texto em qualquer aba ou aplicativo e aperta o atalho definido no teclado.

Após apertar o atalho, o programa irá fazer a tradução e te devolver uma caixa com a tradução na posiçao em que seu mouse se encontra atualmente,
para fechar essa janela basta clicar em qualquer lugar da tela ou na propria caixa.




PARA EXECUTAR ESSE PROGRAMA UTILIZE UMA VENV, NÃO INSTALE OS REQUERIMENTOS NO SEU PYTHON GLOBAL.


COMO UTILIZAR O PROGRAMA:

NA PASTA DO PROGRAMA: ABRA O TERMINAL

NO CMD EXECUTE OS COMANDOS NA SEGUINTE ORDEM:

py -m venv .venv
.\.venv\Scripts\activate.bat
pip install -r requirements.txt
py CodigoFinal.py

NO POWERSHELL EXECUTE OS COMANDOS NA SEGUINTE ORDEM:

py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
py CodigoFinal.py

NO TERMINAL LINUX OU MACOS EXECUTE OS COMANDOS NA SEGUINTE ORDEM:

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 CodigoFinal.py
